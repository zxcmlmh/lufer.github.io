2.修改行内引用代码块颜色

  主题配置文件的颜色选项不能修改行内代码块的颜色，原颜色太淡。

  文件位置\hexo-theme-melody\source\css\var.styl

  修改代码

  本文所做修改已打包至hexo-theme-melody-lufer，可通过npm install hexo-theme-melody-lufer来进行安装。

  使用此包注意要修改_config.yml中的theme为theme: melody-lufer。
